#  Nova AI Chatbot

Nova is an AI-powered chatbot built using LangChain, Mistral AI, and Streamlit.

## Features

- Multiple AI Personalities
  - Normal
  - Funny
  - Angry
  - Sad
- Conversation Memory
- Modern Streamlit Chat Interface
- Clear Chat Functionality
- Secure API Key Management

## Tech Stack

- Python
- LangChain
- Mistral AI
- Streamlit
- Python Dotenv

## Installation

### Clone the Repository

```bash
git clone https://github.com/Ujjvl-hub/Nova-AI-Chatbot.git
cd Nova-AI-Chatbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create .env File

```env
MISTRAL_API_KEY=your_api_key_here
```

### Run the Application

```bash
streamlit run app.py
```
